CSS 基础教程

CSS（Cascading Style Sheets，层叠样式表）负责网页的外观：颜色、字体、间距、布局。HTML 搭骨架，CSS 做装修。

1. 什么是 CSS？

CSS 的基本工作方式是：选择 HTML 元素 → 设置样式属性。

CSS
h1 {
  color: blue;        /* 文字颜色 */
  font-size: 32px;     /* 字体大小 */
  text-align: center;  /* 文字居中 */
}

CSS 语法结构

CSS
选择器 {
  属性名: 属性值;
  属性名: 属性值;
}
  • 选择器：指定要样式化的 HTML 元素
  • 声明块：花括号 { } 内的样式规则
  • 声明：一条 属性: 值; 规则，必须以分号结尾
  • 注释：/* 注释内容 */

层叠（Cascade）的含义

当多个规则作用于同一元素时，浏览器按以下优先级决定最终样式：

  1. !important 声明（尽量避免使用）
  2. 特异性（id > class > 标签）
  3. 来源顺序：后写的规则覆盖先写的

2. 如何引入 CSS

有三种方式将 CSS 应用到 HTML，优先级从低到高：外部样式表 < 内部样式 < 内联样式。

方式一：外部样式表（推荐）

CSS 写在单独的 .css 文件中，通过 <link> 引入，可被多个页面复用。

HTML
<head>
  <link rel="stylesheet" href="css/style.css">
</head>

方式二：内部样式表

HTML
<head>
  <style>
    h1 { color: red; }
  </style>
</head>

方式三：内联样式（不推荐）

HTML
<p style="color: green; font-size: 18px;">绿色文字</p>
💡 最佳实践 日常开发请使用外部样式表。内联样式难以维护，只在极少数动态场景使用。

3. 选择器

选择器决定 CSS 规则应用到哪些元素上，是 CSS 最核心的概念之一。

基础选择器

选择器示例匹配
元素选择器p { }所有 <p> 元素
类选择器.btn { }class="btn" 的元素
ID 选择器#header { }id="header" 的元素
通配符* { }所有元素
CSS
/* 元素选择器 */
p { line-height: 1.6; }

/* 类选择器 — 最常用 */
.highlight { background: yellow; }
.btn.primary { background: blue; }  /* 同时有 btn 和 primary 类 */

/* ID 选择器 — 谨慎使用 */
#logo { width: 120px; }

组合选择器

CSS
/* 后代选择器：nav 内部的所有 a */
nav a { color: #333; }

/* 子选择器：ul 的直接子 li */
ul > li { list-style: none; }

/* 相邻兄弟：h2 后面紧挨的 p */
h2 + p { font-weight: bold; }

/* 分组选择器：h1 和 h2 共用样式 */
h1, h2 { font-family: sans-serif; }

伪类选择器

伪类选择特定状态的元素：

CSS
a:hover { color: red; }           /* 鼠标悬停 */
a:visited { color: purple; }      /* 已访问的链接 */
input:focus { border-color: blue; } /* 获得焦点 */
li:first-child { font-weight: bold; } /* 第一个子元素 */
li:nth-child(2) { color: green; }  /* 第 2 个子元素 */
p:not(.intro) { opacity: 0.8; }    /* 不含 intro 类的 p */

伪元素选择器

CSS
p::before { content: "📌 "; }       /* 元素前插入内容 */
p::after { content: " ✓"; }        /* 元素后插入内容 */
::selection { background: #ff0; }  /* 选中文字的背景 */
input::placeholder { color: #999; } /* 占位符文字 */

4. 颜色与字体

颜色表示方式

CSS
.demo {
  color: red;                    /* 颜色名 */
  color: #ff0000;                /* 十六进制 */
  color: #f00;                   /* 十六进制简写 */
  color: rgb(255, 0, 0);           /* RGB */
  color: rgba(255, 0, 0, 0.5);     /* RGB + 透明度 0~1 */
  color: hsl(0, 100%, 50%);        /* 色相/饱和度/亮度 */
  background-color: #f0f0f0;     /* 背景色 */
}

字体属性

CSS
body {
  font-family: "Microsoft YaHei", sans-serif;  /* 字体族，从左到右回退 */
  font-size: 16px;          /* 字号 */
  font-weight: 400;         /* 粗细：100~900，normal=400, bold=700 */
  font-style: normal;       /* normal | italic 斜体 */
  line-height: 1.6;         /* 行高，无单位时是字号的倍数 */
  text-align: left;         /* left | center | right | justify */
  text-decoration: none;    /* none | underline | line-through */
  text-transform: uppercase; /* 大小写转换 */
  letter-spacing: 1px;      /* 字间距 */
}

字体大小单位

单位说明示例
px像素，绝对单位font-size: 16px;
em相对于父元素字号font-size: 1.5em;
rem相对于根元素（html）字号font-size: 1.25rem;
%百分比font-size: 120%;
vw/vh视口宽度/高度的百分比width: 50vw;
💡 推荐 现代开发中，根元素设 font-size: 16px;，其余用 rem 做字号，便于统一缩放。

5. 盒模型（Box Model）

CSS 中，每个元素都被看作一个矩形盒子，由内容、内边距、边框、外边距四层组成。

margin（外边距）
border（边框）
padding（内边距）
content（内容）
CSS
.box {
  width: 300px;              /* 宽度 */
  height: 200px;             /* 高度 */
  padding: 20px;              /* 内边距：内容与边框的距离 */
  padding: 10px 20px;         /* 上下10 左右20 */
  padding: 10px 20px 30px 40px; /* 上 右 下 左（顺时针） */
  border: 2px solid #ccc;     /* 边框：粗细 样式 颜色 */
  margin: 16px;               /* 外边距：元素与其他元素的距离 */
  margin: 0 auto;           /* 水平居中块级元素 */
  box-sizing: border-box;  /* 宽度包含 padding 和 border */
}
⚠️ 重要 始终设置 box-sizing: border-box;（可放在全局 reset 中）。否则 width: 300px 加上 padding 和 border 后实际宽度会超过 300px。

边框样式

边框样式值：solid（实线）、dashed（虚线）、dotted（点线）、none（无）

背景

CSS
.hero {
  background-color: #f0f0f0;
  background-image: url("bg.jpg");
  background-size: cover;       /* 覆盖整个区域 */
  background-position: center;
  background-repeat: no-repeat;
  border-radius: 8px;         /* 圆角 */
  box-shadow: 0 2px 8px rgba(0,0,0,0.1); /* 阴影 */
}

6. 显示类型与定位

display 属性

值行为典型元素
block独占一行，可设宽高div, p, h1, section
inline行内排列，不可设宽高span, a, strong
inline-block行内排列，但可设宽高button, img
none完全隐藏，不占空间—
flex弹性布局容器—
grid网格布局容器—

position 定位

CSS
/* static — 默认值，正常文档流 */
.normal { position: static; }

/* relative — 相对自身原位置偏移 */
.offset {
  position: relative;
  top: 10px;
  left: 20px;
}

/* absolute — 相对最近的非 static 祖先定位 */
.parent { position: relative; }
.badge {
  position: absolute;
  top: 0;
  right: 0;
}

/* fixed — 相对浏览器窗口固定 */
.navbar {
  position: fixed;
  top: 0;
  width: 100%;
}

/* sticky — 滚动到阈值时"粘"在顶部 */
.sticky-header {
  position: sticky;
  top: 0;
}

z-index 层叠顺序

只对 position 非 static 的元素生效，数值越大越靠前。

7. Flexbox 弹性布局

Flexbox 是现代 CSS 布局的核心工具，特别适合导航栏、卡片排列、垂直居中等场景。

基本概念

  • flex 容器：设置了 display: flex 的父元素
  • flex 项目：容器的直接子元素
  • 主轴：默认水平方向
  • 交叉轴：与主轴垂直的方向
CSS
/* 容器属性 */
.container {
  display: flex;
  flex-direction: row;          /* row | column | row-reverse | column-reverse */
  justify-content: center;     /* 主轴对齐：flex-start | center | space-between | space-around */
  align-items: center;         /* 交叉轴对齐：stretch | center | flex-start | flex-end */
  flex-wrap: wrap;             /* 是否换行：nowrap | wrap */
  gap: 16px;                   /* 项目间距 */
}

/* 项目属性 */
.item {
  flex: 1;                     /* 等分剩余空间 */
  flex-grow: 1;              /* 放大比例 */
  flex-shrink: 0;            /* 不缩小 */
  flex-basis: 200px;           /* 初始大小 */
}

经典布局示例

水平垂直居中

CSS
.center {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

导航栏

CSS
nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 0 24px;
}

8. 响应式设计

让网页在不同屏幕尺寸（手机、平板、桌面）上都能良好显示。

viewport 元标签

HTML
<meta name="viewport" content="width=device-width, initial-scale=1">

媒体查询（Media Queries）

CSS
/* 移动端优先：默认样式给手机 */
.container {
  padding: 16px;
}

.card-grid {
  display: grid;
  grid-template-columns: 1fr;  /* 手机：1 列 */
  gap: 16px;
}

/* 平板：768px 及以上 */
@media (min-width: 768px) {
  .card-grid {
    grid-template-columns: repeat(2, 1fr);  /* 2 列 */
  }
}

/* 桌面：1024px 及以上 */
@media (min-width: 1024px) {
  .card-grid {
    grid-template-columns: repeat(3, 1fr);  /* 3 列 */
  }
  .container {
    max-width: 1200px;
    margin: 0 auto;
  }
}

常用响应式技巧

  • 使用相对单位（%、rem、vw）而非固定像素
  • 图片设置 max-width: 100%; height: auto;
  • 移动端隐藏/显示元素：.mobile-only { display: block; } + 媒体查询切换
  • Flexbox 的 flex-wrap: wrap 让元素自动换行
