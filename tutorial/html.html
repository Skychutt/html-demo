HTML 基础教程

HTML（HyperText Markup Language，超文本标记语言）是构建网页的基础。它用「标签」来描述页面的结构和内容，浏览器读取 HTML 后将其渲染成你看到的网页。

1. 什么是 HTML？

想象一下，你在盖一座房子：HTML 就是房子的框架（墙、门、窗的位置），CSS 是装修（颜色、家具），JavaScript 是电路和水管（让 things 动起来）。

HTML 能做什么？

  • 定义页面的标题、段落、图片、链接等内容
  • 创建表单，让用户输入信息（登录、搜索、注册）
  • 组织内容的层级结构（标题、列表、表格）
  • 为 CSS 和 JavaScript 提供操作的目标

HTML 不能做什么？

  • 不能直接控制颜色、字体大小（那是 CSS 的工作）
  • 不能实现点击按钮弹出对话框（那是 JavaScript 的工作）
  • 不是编程语言，没有 if/else、循环等逻辑
💡 小提示 HTML 文件以 .html 为扩展名，用浏览器直接打开就能查看效果。你也可以在编辑器里写 HTML，保存后用浏览器打开预览。

2. HTML 文档结构

每个 HTML 页面都遵循一个固定的基本结构。下面是一个最简模板：

HTML 
<!-- 文档类型声明：告诉浏览器这是 HTML5 文档 -->
<!doctype html>

<!-- html 是根元素，lang 属性指定页面语言 -->
<html lang="zh-CN">

  <!-- head：存放页面的元信息，用户看不到 -->
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>我的第一个网页</title>
  </head>

  <!-- body：存放用户可见的所有内容 -->
  <body>
    <h1>你好，世界！</h1>
    <p>这是我的第一个 HTML 页面。</p>
  </body>

</html>

各部分说明

部分 作用
<!doctype html> 文档类型声明，必须是第一行，表示使用 HTML5 标准
<html> 根元素，包裹整个页面。lang="zh-CN" 表示中文，有利于 SEO 和屏幕阅读器
<head> 头部区域：标题、字符编码、CSS/JS 引用、SEO 信息等
<meta charset="utf-8"> 字符编码，支持中文等所有语言字符，避免乱码
<title> 页面标题，显示在浏览器标签页上
<body> 主体区域：所有用户能看到的内容都写在这里
📝 注释 HTML 注释写法是 <!-- 注释内容 -->，浏览器不会显示注释，但开发者可以在源码中看到，用于说明代码。

3. 标签、元素与属性

标签（Tag）

HTML 用尖括号包裹的标签来标记内容。大多数标签成对出现：

  • 开始标签：<p>
  • 结束标签：</p>（注意有斜杠 /）
  • 标签之间的内容：段落文字

三者合在一起叫做一个元素（Element）：<p>段落文字</p>

自闭合标签

有些标签不需要结束标签，称为空元素或自闭合标签：

HTML
<img src="photo.jpg" alt="照片">
<br>       <!-- 换行 -->
<hr>       <!-- 水平分割线 -->
<input type="text">
<meta charset="utf-8">

属性（Attribute）

属性写在开始标签里，为元素提供额外信息，格式为 属性名="属性值"：

HTML
<a href="https://example.com" target="_blank">访问网站</a>
<img src="logo.png" alt="网站 Logo" width="200">
<div id="main" class="container">...</div>

常用全局属性

属性说明示例
id唯一标识符，整个页面只能有一个相同 idid="header"
class类名，可重复使用，用于 CSS 样式和 JS 选择class="btn primary"
style内联样式（不推荐大量使用）style="color: red;"
title鼠标悬停时显示的提示文字title="点击查看详情"
hidden隐藏元素hidden
⚠️ 注意 属性值始终用双引号包裹。标签名和属性名不区分大小写，但推荐使用小写。

4. 文本标签

HTML 提供了丰富的标签来组织文本内容，正确使用标题标签对 SEO 和可访问性很重要。

标题 h1 ~ h6

h1 最大最重要（通常每页只有一个），h6 最小。不要跳级使用（如 h1 后直接 h3）。

5. 链接与图片

超链接 <a>

链接是 Web 的核心。href 属性指定目标地址：

HTML
<!-- 外部链接 -->
<a href="https://www.baidu.com">访问百度</a>

<!-- 新标签页打开 -->
<a href="https://github.com" target="_blank" rel="noopener">GitHub</a>

<!-- 页面内锚点跳转 -->
<a href="#section2">跳到第二节</a>
<h2 id="section2">第二节</h2>

<!-- 相对路径：同目录下的 about.html -->
<a href="about.html">关于我们</a>

<!-- 邮件链接 -->
<a href="mailto:hello@example.com">发送邮件</a>

图片 <img>

HTML
<img src="images/photo.jpg" alt="一张风景照片" width="400" height="300">
  • src：图片路径（必填）
  • alt：替代文字，图片加载失败或屏幕阅读器时使用（必填，SEO 友好）
  • width / height：尺寸，建议设置以防止页面布局跳动
🚫 重要 永远不要省略 alt 属性。装饰性图片可以写 alt=""，但必须有这个属性。

7. 表格

表格用于展示结构化数据（如成绩单、价格表），不要用来做页面布局。

HTML
<table>
  <thead>
    <tr>
      <th>姓名</th>
      <th>科目</th>
      <th>分数</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>张三</td>
      <td>数学</td>
      <td>95</td>
    </tr>
    <tr>
      <td>李四</td>
      <td>英语</td>
      <td>88</td>
    </tr>
  </tbody>
</table>
标签含义
table表格容器
thead表头区域
tbody表体区域
tr表格行（table row）
th表头单元格（table header），默认加粗居中
td数据单元格（table data）
colspan跨列合并，如 colspan="2"
rowspan跨行合并，如 rowspan="2"

8. 表单

表单让用户向网站提交数据：登录、搜索、注册、留言等。

HTML
<form action="/submit" method="post">

  <label for="username">用户名：</label>
  <input type="text" id="username" name="username" placeholder="请输入用户名" required>

  <label for="password">密码：</label>
  <input type="password" id="password" name="password" required>

  <label for="email">邮箱：</label>
  <input type="email" id="email" name="email">

  <label for="bio">简介：</label>
  <textarea id="bio" name="bio" rows="4"></textarea>

  <label>性别：</label>
  <input type="radio" name="gender" value="male" id="male"> <label for="male">男</label>
  <input type="radio" name="gender" value="female" id="female"> <label for="female">女</label>

  <label>爱好：</label>
  <input type="checkbox" name="hobby" value="reading"> 阅读
  <input type="checkbox" name="hobby" value="coding"> 编程

  <label for="city">城市：</label>
  <select id="city" name="city">
    <option value="">请选择</option>
    <option value="beijing">北京</option>
    <option value="shanghai">上海</option>
  </select>

  <button type="submit">提交</button>
  <button type="reset">重置</button>

</form>

常用 input 类型

type 值说明
text单行文本
password密码（输入内容被隐藏）
email邮箱（浏览器会验证格式）
number数字
date日期选择器
checkbox复选框（多选）
radio单选按钮（同 name 为一组）
file文件上传
hidden隐藏字段
💡 无障碍提示 每个表单控件都应该有对应的 <label>，通过 for 属性与控件的 id 关联。这样点击标签文字也能聚焦到输入框。

9. 语义化标签（HTML5）

HTML5 引入了语义化标签，让代码结构更清晰，也对搜索引擎和屏幕阅读器更友好。

HTML
<header>
  <nav>
    <a href="/">首页</a>
    <a href="/about">关于</a>
  </nav>
</header>

<main>
  <article>
    <h1>文章标题</h1>
    <p>文章正文内容...</p>
  </article>

  <aside>
    <h2>相关链接</h2>
    <ul>...</ul>
  </aside>
</main>

<footer>
  <p>&copy; 2026 我的网站</p>
</footer>
标签语义
header页头 / 区块头部
nav导航链接区域
main页面主要内容（每页唯一）
article独立的文章内容
section文档中的主题分区
aside侧边栏 / 附属内容
footer页脚 / 区块底部
figure + figcaption图片/图表及其说明

通用容器

  • <div>：块级容器，无语义，用于分组和布局
  • <span>：行内容器，无语义，用于包裹文字片段

10. 最佳实践与常见错误

✅ 推荐做法

  • 始终写 <!doctype html> 和 <meta charset="utf-8">
  • 使用语义化标签，而不是全部用 <div>
  • 图片必须有 alt 属性
  • 表单控件配 <label>
  • 保持代码缩进整齐，便于阅读和维护
  • 标签名、属性名用小写
  • 属性值用双引号

❌ 常见错误

  • 忘记闭合标签（如 <p> 没有 </p>）
  • 用表格做页面布局（已过时，用 CSS Flexbox/Grid）
  • 多个 h1 标签（通常每页只用一个）
  • 跳过标题级别（h1 后直接 h4）
  • 用 <br> 做间距（应该用 CSS margin/padding）

特殊字符转义

在 HTML 中，某些字符需要转义：

字符转义写法说明
<&lt;小于号
>&gt;大于号
&&amp;和号
"&quot;双引号
