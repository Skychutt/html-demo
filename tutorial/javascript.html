JavaScript 基础教程

JavaScript（简称 JS）是 Web 的编程语言，让网页具备交互能力：响应点击、验证表单、动态更新内容、与服务器通信等。

1. 什么是 JavaScript？

继续用盖房子的比喻：HTML 是框架，CSS 是装修，JavaScript 是电路和智能系统——灯能开关、门铃能响、温度能自动调节。

JavaScript 能做什么？

  • 响应用户操作（点击按钮、输入文字、滚动页面）
  • 动态修改页面内容和样式
  • 验证表单数据
  • 创建动画和特效
  • 与服务器通信（发送/接收数据）
  • 存储数据到浏览器（localStorage）

JavaScript 在哪里运行？

  • 浏览器中：前端开发的主要场景（Chrome、Firefox、Edge 都内置 JS 引擎）
  • Node.js：在服务器端运行 JS（后端开发）
💡 调试工具 按 F12 打开浏览器开发者工具，切换到「Console（控制台）」标签页，可以直接输入 JS 代码并查看输出。学习 JS 时这是你的好帮手。

2. 如何引入 JavaScript

方式一：外部脚本（推荐）

HTML
<!-- 放在 </body> 前，确保 HTML 先加载 -->
<script src="js/app.js"></script>
</body>

方式二：内部脚本

HTML
<script>
  console.log('Hello, JavaScript!');
</script>
⚠️ 注意 使用 src 属性的 <script> 标签内不能再写代码。脚本通常放在 </body> 前，避免阻塞页面渲染。

3. 变量与数据类型

声明变量

JavaScript
let name = '小明';       // 可变变量（推荐）
const PI = 3.14159;       // 常量，不可重新赋值（推荐）
var oldStyle = '旧写法';  // 避免使用 var

name = '小红';            // ✅ let 可以重新赋值
// PI = 3.14;             // ❌ const 不能重新赋值，会报错

基本数据类型

类型说明示例
String 字符串文本'你好' 或 "世界"
Number 数字整数或小数42, 3.14
Boolean 布尔真或假true, false
undefined未定义/未赋值undefined
null空值null
Symbol唯一标识符（进阶）—
BigInt大整数（进阶）9007199254740991n

引用类型

JavaScript
const arr = [1, 2, 3];           // 数组
const person = { name: '张三', age: 25 };  // 对象
const greet = function() { ... };  // 函数也是对象

类型检测与转换

JavaScript
typeof 'hello';     // "string"
typeof 42;          // "number"
typeof true;         // "boolean"

Number('123');      // 123（字符串转数字）
String(456);        // "456"（数字转字符串）
Boolean(0);         // false（0、""、null、undefined、NaN 都为 false）
parseInt('42px');   // 42（解析整数）
parseFloat('3.14'); // 3.14

模板字符串

JavaScript
const name = '小明';
const age = 18;

// 用反引号 ` 包裹，${} 插入变量或表达式
const message = `我叫${name}，今年${age}岁，明年${age + 1}岁。`;
console.log(message);  // 我叫小明，今年18岁，明年19岁。

4. 运算符

算术运算符

JavaScript
10 + 3;   // 13  加
10 - 3;   // 7   减
10 * 3;   // 30  乘
10 / 3;   // 3.333...  除
10 % 3;   // 1   取余（模）
2 ** 3;   // 8   幂运算

let count = 0;
count++;      // 自增，等同于 count = count + 1
count--;      // 自减
count += 5;   // 等同于 count = count + 5

比较运算符

JavaScript
5 == '5';   // true  （宽松相等，会类型转换）
5 === '5';  // false （严格相等，类型和值都要相同）✅ 推荐
5 != '5';   // false
5 !== '5';  // true
10 > 5;    // true
10 >= 10;  // true

逻辑运算符

JavaScript
true && false;   // false  与（AND）：都为 true 才 true
true || false;   // true   或（OR）：有一个 true 就 true
!true;          // false  非（NOT）：取反

// 短路求值
const name = userName || '访客';  // userName 为空时用默认值
const city = user?.address?.city;       // 可选链，安全访问嵌套属性

5. 条件语句

根据条件决定执行不同的代码分支。

if / else if / else

JavaScript
const score = 85;

if (score >= 90) {
  console.log('优秀');
} else if (score >= 60) {
  console.log('及格');
} else {
  console.log('不及格');
}
// 输出：及格

switch

JavaScript
const day = 3;

switch (day) {
  case 1:
    console.log('星期一');
    break;
  case 2:
    console.log('星期二');
    break;
  case 3:
    console.log('星期三');
    break;
  default:
    console.log('其他日子');
}

三元运算符

JavaScript
const age = 20;
const status = age >= 18 ? '成年人' : '未成年人';
// 条件 ? 真值 : 假值

6. 循环

重复执行一段代码，直到满足退出条件。

for 循环

JavaScript
// 经典 for：初始化; 条件; 更新
for (let i = 0; i < 5; i++) {
  console.log(`第 ${i} 次循环`);
}
// 输出：第 0 次循环 ... 第 4 次循环

while 和 do...while

JavaScript
let n = 0;
while (n < 3) {
  console.log(n);
  n++;
}

do {
  console.log('至少执行一次');
} while (false);

遍历数组

JavaScript
const fruits = ['苹果', '香蕉', '橙子'];

// for...of（推荐）
for (const fruit of fruits) {
  console.log(fruit);
}

// forEach 方法
fruits.forEach(function(fruit, index) {
  console.log(index, fruit);
});

// break 跳出循环，continue 跳过本次

7. 函数

函数是可复用的代码块，接收输入（参数），返回输出（返回值）。

函数声明

JavaScript
// 函数声明
function greet(name) {
  return `你好，${name}！`;
}

// 函数表达式
const add = function(a, b) {
  return a + b;
};

// 箭头函数（ES6，最常用）
const multiply = (a, b) => a * b;
const square = x => x * x;
const sayHi = () => console.log('Hi!');

console.log(greet('小明'));  // 你好，小明！
console.log(add(2, 3));       // 5

默认参数与剩余参数

JavaScript
function createUser(name, role = '用户') {
  return { name, role };
}
createUser('张三');           // { name: '张三', role: '用户' }
createUser('李四', '管理员');  // { name: '李四', role: '管理员' }

function sum(...numbers) {
  return numbers.reduce((total, n) => total + n, 0);
}
sum(1, 2, 3, 4);  // 10

作用域

  • 全局作用域：在函数外声明的变量，整个脚本都能访问
  • 函数作用域：在函数内声明的变量，只在函数内有效
  • 块级作用域：let 和 const 在 { } 块内有效

8. 数组与对象

数组（Array）

JavaScript
const colors = ['红', '绿', '蓝'];

// 访问和修改（索引从 0 开始）
colors[0];           // '红'
colors[1] = '黄';     // 修改
colors.length;       // 3

// 常用方法
colors.push('紫');       // 末尾添加 → ['红','黄','蓝','紫']
colors.pop();          // 末尾删除 → '紫'
colors.unshift('黑');    // 开头添加
colors.shift();        // 开头删除
colors.includes('红');  // true
colors.indexOf('蓝');    // 2

// 不改变原数组的方法
const doubled = [1, 2, 3].map(n => n * 2);       // [2, 4, 6]
const evens = [1, 2, 3, 4].filter(n => n % 2 === 0); // [2, 4]
const total = [1, 2, 3].reduce((sum, n) => sum + n, 0); // 6
const joined = ['a', 'b'].join('-');          // 'a-b'

对象（Object）

JavaScript
const user = {
  name: '张三',
  age: 25,
  email: 'zhangsan@example.com',
  greet() {
    return `我是${this.name}`;
  }
};

// 访问属性
user.name;           // 点语法
user['age'];         // 方括号语法
user.greet();        // 调用方法

// 添加/修改/删除
user.city = '北京';
delete user.email;

// 解构赋值
const { name, age } = user;
const { name: userName } = user;  // 重命名

// 遍历对象
for (const key in user) {
  console.log(key, user[key]);
}
Object.keys(user);    // ['name', 'age', 'city', 'greet']
Object.values(user);  // ['张三', 25, '北京', function]

JSON

JavaScript
// JS 对象 ↔ JSON 字符串（常用于数据传输和存储）
const obj = { name: '小明', scores: [90, 85] };
const json = JSON.stringify(obj);   // '{"name":"小明","scores":[90,85]}'
const parsed = JSON.parse(json);    // 还原为对象

9. DOM 操作

DOM（Document Object Model）是浏览器将 HTML 文档解析成的树形结构。JavaScript 通过 DOM API 来查找、修改、创建和删除页面元素。

选择元素

JavaScript
// 选择一个元素
document.getElementById('header');
document.querySelector('.btn');          // 第一个匹配 CSS 选择器的
document.querySelector('nav a.active');

// 选择多个元素
document.querySelectorAll('li');       // NodeList
document.getElementsByClassName('item');
document.getElementsByTagName('p');

修改内容和样式

JavaScript
const title = document.querySelector('h1');

// 文本内容
title.textContent = '新标题';       // 纯文本
title.innerHTML = '<em>新标题</em>';  // 可含 HTML（注意 XSS 风险）

// 属性
const link = document.querySelector('a');
link.href = 'https://example.com';
link.setAttribute('target', '_blank');
link.getAttribute('href');

// CSS 样式
title.style.color = 'red';
title.style.fontSize = '24px';

// CSS 类（推荐）
title.classList.add('active');
title.classList.remove('hidden');
title.classList.toggle('highlight');
title.classList.contains('active');  // true/false

创建和删除元素

JavaScript
// 创建
const li = document.createElement('li');
li.textContent = '新项目';
li.classList.add('list-item');

// 添加到 DOM
const list = document.querySelector('ul');
list.appendChild(li);          // 末尾添加
list.prepend(li);              // 开头添加
list.insertBefore(li, refNode); // 指定位置

// 删除
li.remove();                   // 现代写法
list.removeChild(li);          // 传统写法

10. 事件处理

事件是用户或浏览器触发的动作：点击、输入、滚动、键盘按键、页面加载等。

添加事件监听

JavaScript
const btn = document.querySelector('#myBtn');

// 推荐：addEventListener
btn.addEventListener('click', function(event) {
  console.log('按钮被点击了！');
  console.log(event.target);  // 触发事件的元素
});

// 箭头函数写法
btn.addEventListener('click', () => {
  alert('Hello!');
});

// 移除监听器
function handleClick() { ... }
btn.addEventListener('click', handleClick);
btn.removeEventListener('click', handleClick);

常用事件

事件触发时机
click鼠标点击
dblclick双击
mouseenter / mouseleave鼠标进入/离开元素
keydown / keyup键盘按下/抬起
input输入框内容变化
change输入框失去焦点且值改变
submit表单提交
scroll元素滚动
load页面或资源加载完成
DOMContentLoadedHTML 解析完成（不等图片）

事件对象常用属性和方法

JavaScript
element.addEventListener('click', (e) => {
  e.preventDefault();    // 阻止默认行为（如链接跳转、表单提交）
  e.stopPropagation(); // 阻止事件冒泡
  e.target;              // 实际被点击的元素
  e.currentTarget;       // 绑定监听器的元素
});

// 表单提交示例
form.addEventListener('submit', (e) => {
  e.preventDefault();  // 阻止页面刷新
  const data = new FormData(form);
  console.log(data.get('username'));
});

事件委托

利用事件冒泡，在父元素上统一处理子元素的事件，适合动态添加的元素：

JavaScript
list.addEventListener('click', (e) => {
  if (e.target.tagName === 'LI') {
    console.log('点击了：', e.target.textContent);
  }
});

11. 异步入门

JavaScript 是单线程的，但遇到耗时操作（网络请求、定时器）不会阻塞页面，靠的是异步机制。

setTimeout 和 setInterval

JavaScript
// 延迟执行（毫秒）
const timerId = setTimeout(() => {
  console.log('3 秒后执行');
}, 3000);
clearTimeout(timerId);  // 取消

// 重复执行
const intervalId = setInterval(() => {
  console.log('每秒执行');
}, 1000);
clearInterval(intervalId);  // 停止

Promise

JavaScript
const promise = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve('成功的结果');
    // reject(new Error('失败原因'));
  }, 1000);
});

promise
  .then(result => console.log(result))
  .catch(error => console.error(error));

async / await（现代写法）

JavaScript
async function fetchUser() {
  try {
    const response = await fetch('https://api.example.com/user');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('请求失败：', error);
  }
}

localStorage 本地存储

JavaScript
// 存储（值必须是字符串，对象需 JSON.stringify）
localStorage.setItem('username', '小明');
localStorage.setItem('settings', JSON.stringify({ theme: 'dark' }));

// 读取
localStorage.getItem('username');
JSON.parse(localStorage.getItem('settings'));

// 删除
localStorage.removeItem('username');
localStorage.clear();  // 清空全部
