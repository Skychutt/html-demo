前端开发零基础入门

一套完整、详细、面向新手的 HTML、CSS、JavaScript 和 TypeScript 基础语法教程。
跟着学，你就能做出自己的第一个网页。

📄

HTML 基础

学习网页的骨架结构：标签、元素、属性、文本、链接、图片、列表、表格、表单与语义化标签。

10 个章节 · 建议 2-3 小时
🎨

CSS 基础

学习网页的样式美化：选择器、颜色字体、盒模型、布局、Flexbox、定位与响应式设计。

9 个章节 · 建议 3-4 小时
⚡

JavaScript 基础

学习网页的交互逻辑：变量、数据类型、条件循环、函数、数组对象、DOM 操作与事件处理。

11 个章节 · 建议 5-6 小时
🔷

TypeScript 基础

在 JS 基础上学习静态类型：类型注解、接口、泛型、与 JavaScript 的区别及工程化实践。

15 个章节 · 建议 4-6 小时

推荐学习路线

  1. HTML 基础 — 先理解网页由什么构成，学会写基本的页面结构
  2. CSS 基础 — 给页面添加颜色、布局和美观的样式
  3. JavaScript 基础 — 让页面"活"起来，响应用户的点击和操作
  4. TypeScript 基础 — 给 JS 加上类型系统，为大型项目和框架开发打基础
  5. 动手练习 — 尝试做一个简单的个人主页或待办事项页面

你需要准备什么？

  • 一个代码编辑器：推荐 VS Code、WebStorm 或 Cursor（你现在用的就可以）
  • 一个现代浏览器：Chrome、Edge 或 Firefox 均可
  • 无需安装任何框架：本教程全部使用原生 HTML、CSS、JavaScript
